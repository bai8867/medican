# medican 仓库目录说明与注释树

本文描述**常用目录的职责**与**精简目录树**；行尾 `# …` 为注释（不是路径的一部分）。大体积或可再生产物（如 `node_modules/`、`target/`、`dist/`、`.git/`）不展开。

更完整的产品与交付说明见 [`handbook/README.md`](handbook/README.md) 及根目录 [`README.md`](../README.md)。

---

## 1. 顶层一览（带注释）

```text
medican/                                    # 仓库根：README / AGENTS、Windows 一键脚本、Python API 联调
├── .github/                               # CI：后端测试、前端 build、供应链 audit、PR 模板
├── campus-diet-backend/                   # Spring Boot 2.7 + Java 11；默认 http://127.0.0.1:11888
├── config/                                # 与部署相关的共享配置片段（若有）
├── docs/                                  # 工程文档：未完成项、部署、API、可观测、安全基线
│   ├── handbook/                          # 产品/交付/答辩类 Markdown；exports/ 为 docx/html
│   ├── superpowers/                       # 架构升级类计划与设计留档
│   └── …                                  # 其余见下文「docs/ 精要」
├── McpTools/                              # MCP 相关工具与实验目录
├── observability/                         # Prometheus / Grafana / Alertmanager 等 compose 与配置
├── plugins/                               # Cursor 插件示例（与主应用解耦）
├── release/                               # Build-MedicanDeliveryZip 输出（默认 .gitignore）
├── scripts/                               # PowerShell / Node / Python 运维与数据脚本
├── skills-repos/                          # 第三方 skills 镜像（体积大；按需保留或改 gitignore）
├── tcm-diet-frontend/                     # Vue 3 + Vite 6；开发默认 http://localhost:11999
├── templates/                             # 脚手架与模板（如 cursor-tools 场景）
├── AGENTS.md                              # 智能体/工具链约定、端口与脚本索引
├── README.md                              # 人类入口：启动顺序、文档表、环境变量速查
├── docker-compose.mysql-dev.yml           # 可选：容器化 MySQL 开发（与 bat 二选一）
├── maven-settings-d.xml                   # 与后端 `mvn -s` 一致的本地仓库配置
├── mysql授权tcm_app.bat                   # 双击：本机 MySQL 对 tcm_app 授权（调 PowerShell）
├── test_integration.py                    # 后端 API 轻量联调（标准库；路径勿随意移动）
├── run_test_suite.py                      # 聚合：前端单测 + test_integration 报告
└── …                                      # 其它 bat、本机 mysql ini 等见根目录列表
```

## 2. 后端 `campus-diet-backend/`（精要）

```text
campus-diet-backend/
├── .mvn/wrapper/                          # Maven Wrapper（仅 script 模式时需网络拉发行包）
├── config/                                # 数据源 override 示例（含 mysql 示例 yml）
├── scripts/                               # 后端侧一次性脚本（如 dev-bootstrap）
├── src/main/java/                         # 业务域、REST、安全过滤器、启动 Runner
├── src/main/resources/
│   ├── application*.yml                   # 多 profile：默认 h2；mysql/mock-import 等
│   ├── bootstrap/*.json                   # 种子菜谱、场景食疗等数据
│   └── db/*.sql                           # schema、data、增量 patch（由 sql init 策略控制）
├── src/test/java/                         # JUnit / MockMvc / 契约与集成测
├── pom.xml
├── mvnw / mvnw.cmd                        # 无全局 Maven 时使用
└── target/                                # 构建产物（忽略）：classes、test-classes、可执行 jar
```

## 3. 前端 `tcm-diet-frontend/`（精要）

```text
tcm-diet-frontend/
├── e2e/                                   # Playwright 规格（含 CI gate / smoke）

├── dist/                                  # npm run build 产出（忽略）；勿提交

```

## 5. `scripts/`（常见脚本）

| 脚本 | 注释 |
|------|------|
| `Use-RepoJavaMaven.ps1` | 统一 JDK11 + Maven 路径，供其它脚本 dot-source |
| `Run-BackendTests.ps1` | 一键后端测试（含安全基线相关用例） |
| `Build-MedicanDeliveryZip.ps1` | 打离线交付包至 `release/` |
| `Start-MysqlDev.ps1` / `Run-MysqlGrantTcmApp.ps1` | 本机 MySQL 启停与授权 |
| `dev-backend-probe.mjs` | 前端 `npm run dev` 前探活后端 |

---

## 6. 其它目录一句话

| 路径 | 注释 |
|------|------|
| `observability/` | 与 `docs/observability-*.md` 配套的 compose、Prometheus、Grafana 配置 |
| `plugins/` | Cursor 插件示例（`my-plugin`），不参与主站构建 |
| `McpTools/` | MCP 实验与工具集合 |
| `templates/` | 从模板生成 `.cursor` 等开发配置的参考 |
| `skills-repos/` | 上游 skills 仓库与 zip 缓存；clone 体积大，是否长期进 Git 请自行评估 |
| `config/`（根级） | 与多模块部署相关的共享配置（若有） |
